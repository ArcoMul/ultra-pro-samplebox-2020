# Ultra Pro Samplebox 2020

A self build samplebox / loopmachine using:

- Raspberry Pi
- Breadboard
- Buttons
- Aux cable
- USB Microphone

## Start

`$ python3 start.py`

**start.py**  
Starts the application

**detect.py**  
Detects which audio devices are connected, to find the right index to be filled in audio.py
